CSS first steps

Getting started with CSS

  1. What is the following called, and what are its various parts called?

    h1 {
  color: red;
  font-size: 5em;
}

    The whole thing is the rule. The h1 is the selector. color: red and font-size: 5em are declarations. color and font-size are properties. red and 5em are the values.

How CSS is structured

  1. What are the three ways of applying CSS to HTML, from best to worst?

    External stylesheet
    <link rel="stylesheet" href="styles.css"> in the head
    Internal stylesheet
    putting CSS between <style></style> tags in the head
    Inline styles
    putting CSS in a style attribute on an element

  2. What are cascade and specificity?

    The cascade means that later styles take precedence over previous styles. Specificity means that the more specifically selected styles take precedence over less specific ones.

  3. Are CSS properties and values case-sensitive?

    Yes.

  4. What happens if the browser encounters CSS it doesn't understand?

    The whole declaration is ignored.

  5. Which country's spelling standard does CSS use?

    The United States

  6. How would you use an @rule to import a stylesheet into another CSS stylesheet?

    @import 'styles2.css'

  7. What happens when you omit values in CSS shorthand?

    Those values get reset.

  8. How do you make comments in CSS?

    /* This is a comment */

How CSS works

  1. How does a browser basically load a page, in steps?

    1. It loads and parses the HTML.
    2. It creates a DOM (Document Object Model).
    3. It loads and parses the CSS.
    4. It attaches the CSS to DOM nodes (render tree).
    5. It displays the webpage on the screen (painting).

  2. What is the DOM?

    DOM stands for the Document Object Model. It represents the document in the computer's memory, and is like a tree where every element, attribute, and piece of text becomes a node on the tree, with parent nodes, child nodes, and sibling nodes.

CSS building blocks

Cascade and inheritance

  1. What is inheritance?

    Whether or not a property inherits a value from the parent element.

  2. What property values control inheritance?

    • inherit
    • initial (the property's initial value, as defined in the CSS specifications; different from the browser's default styles)
    • unset (sets the inheritance to however the property acts by default, inherit, or initial)

  3. What factors contribute to which rule will be applied in the case of conflict, in order of increasing importance?

    1. Source order
    2. Specificity
    3. Importance

  4. What things don't impact specificity?

    The universal selector (*), combinators, and the negation pseudo-class (:not)

  5. What system can be used to determine specificity?

    A scoring system which works as follows:

    1. Thousands: Score one in this column if it's an inline style.
    2. Hundreds: Score one for each ID selector in the overall selector.
    3. Tens: Score one for each class, attribute and pseudo-class in the overall selector.
    4. Ones: Score one for each element or pseudo-element in the overall selector.

  6. What is importance? When should you use it?

    When you put !important next to a property value, you mark that declaration as important (e.g. border: none !important). You should only use it when absolutely necessary, because it messes with the rules of the cascade.

  7. How are conflicting declarations applied across different stylesheets?

    In order of increasing importance:

    1. Declarations in user agent stylesheets (browser default styles)
    2. Normal declarations in user stylesheets (custom styles set by the user)
    3. Normal declarations in author stylesheets (styles set by web devs)
    4. Important declarations in author stylesheets
    5. Important declarations in user stylesheets

Type, class, and ID selectors

  1. Write the selector that targets all <span></span> elements.

    span

  2. Write the selector that targets every single thing in the document.

    *

  3. How can you use a universal selector to make a selector easier to read?

    Instead of something like article :first-child (which could be confused with article:first-child), you can write article *:first-child.

  4. Write the selector that targets all elements with a class of cereal.

    .cereal

  5. Write the selector that targets all <span></span>s with a class of cereal.

    span.cereal

  6. Write the selector that targets all elements with a class of cereal and a class of milk.

    .cereal.milk

  7. Write the selector that targets all elements with an id of shoop.

    #shoop

Attribute selectors

  1. Write the selector that targets all <a></a>s with a title attribute.

    a[title]

  2. Write the selector that targets all <a></a>s with a href attribute with a value of exactly http://example.com.

    a[href="http://example.com"]

  3. Write the selector that targets all <p></p>s with a class attribute, of which special is either its exact value or one of its values (in a space-separated list).

    p[class~="special"]

  4. Write the selector that targets all <div></div>s with a class attribute, whose value is either exactly zh or begins with zh-.

    div[lang|="zh"]

  5. Write the selector that targets all <li></li>s with a class attribute whose value begins with box-.

    li[class^="box-"]

  6. Write the selector that targets all <li></li>s with a class attribute whose value ends with -box.

    li[class$="-box"]

  7. Write the selector that targets all <li></li>s with a class attribute whose value contains box anywhere in the string.

    li[class*="box"]

  8. How do you tell the browser to match an attribute value case-insensitively?

    Put an i next to the attribute, for example, li[class*="box" i]

Pseudo-classes and pseudo-elements

  1. How are pseudo-classes denoted?

    With a :

  2. Write the selector that targets every <p></p> that is the first element in an <article></article>.

    article p:first-child

  3. Write the selector that targets every <p></p> that is the last element in an <article></article>.

    article p:last-child

  4. Write the selector that targets every <p></p> that is the only element in an <article></article>.

    article p:only-child

  5. Write the selector that targets every <input> that fails to validate.

    input:invalid

  6. Write the selector that targets every <a></a> that is being hovered over.

    a:hover

  7. Write the selector that targets every <a></a> that is being focused using keyboard controls.

    a:focus

  8. How are pseudo-elements denoted?

    With a ::

  9. Write the selector that targets the first line of every <p></p> in an <article></article>.

    article p::first-line

  10. Write the rule that puts the string --> before every element with a class of box.

    .box::before {content: "-->";}

  11. Write the rule that puts the string --> after every element with a class of box.

    .box::after {content: "-->";}

Combinators

  1. Write the selector that targets every <p></p> inside an <article></article> inside a <section></section>. What kind of combinator is this?

    section article p

    This is a descendant combinator.

  2. Write the selector that targets every <p></p> that is a direct child of an <article></article>. What kind of combinator is this?

    article > p

    This is a child combinator.

  3. Write the selector that targets every <img> that comes right after a <p></p> at the same hierarchy level. What kind of combinator is this?

    p + img

    This is an adjacent sibling combinator.

  4. Write the selector that targets every <img> that comes anywhere after a <p></p> at the same hierarchy level. What kind of combinator is this?

    p ~ img

    This is a general sibling combinator.

The box model

  1. Do inline boxes respect width and height attributes?

    No.

  2. Do padding, margins, and borders on inline boxes push away other inline boxes?

    Vertically, no. Horizontally, yes.

  3. What is the difference between inner and outer display types?

    An outer display type (e.g. block and inline) determines how the box interacts with other boxes around it. An inner display type (e.g. flex and grid) determines how elements inside the box interact with one another.

  4. What are the parts of a CSS box, from inside to outside?

    Content, padding, border, and margin.

  5. What is the difference between the standard box model and the alternative box model?

    In the alternative box model, width and height include the padding and border.

  6. How do you make all elements follow the alternative box model?

    html {box-sizing: border-box;}
*, *::before, *::after {box-sizing: inherit;}

  7. If one element with a margin-top of 30px is right below another element with a margin-bottom of 50px, what will be the margin between the two elements? What is this called?

    50px. This is called margin collapsing.

  8. What is, say, border-left shorthand for?

    border-left-width, border-left-style, and border-left-color

  9. What is display: inline-block for?

    For inline boxes that respect width and height, and whose padding, margins, and borders push away other elements in all directions.

Backgrounds and borders

  1. Write the declaration that creates a background image with balloons.png.

    background-image: url('balloons.png')

    1. What would you add so it doesn't repeat?

      background-repeat: no-repeat

    2. So it only repeats horizontally?

      background-repeat: repeat-x

    3. So it only repeats vertically?

      background-repeat: repeat-y

    4. So it repeats in both directions?

      background-repeat: repeat

    5. What would you add so it completely covers the box area while maintaining its aspect ratio?

      background-size: cover

    6. So it fits inside the box while maintaining its aspect ratio?

      background-size: contain

  2. What property is used to position a background image?

    background-position

    1. How does the coordinate system work with this?

      The top-left is the origin. The first coordinate is the horizontal position; the second coordinate is the vertical position.

    2. What keywords does this take?

      top, left, bottom, right, and center

    3. What happens when you mix a keyword with a length/percentage?

      The result means the distance from that keyword. For example, top 20px right 10px means 20px from the top and 10px from the right.

  3. Write the declaration that fixes an element's background to the page (scrolls when the page is scrolled, stays fixed when the element's content is scrolled).

    background-attachment: scroll

  4. Write the declaration that fixes an element's background to the viewport (stays fixed whether the page or the element's content is scrolled; always remains in the same position on the screen).

    background-attachment: fixed

  5. Write the declaration that fixes an element's background to the element itself (scrolls whether the page or the element's content is scrolled, unless the element doesn't scroll on the page, like if it has position: fixed)

    background-attachment: local

  6. What is the rule with specifying background-color using the background shorthand property?

    It must come after the last comma.

  7. What is the rule with specifying background-size using the background shorthand property?

    It can only come immediately after background-position, separated by a forward slash (e.g. center/80%).

  8. How do you round the corners of a border?

    Use the border-radius property. The first value is the horizontal radius, and the second is the vertical radius.

Handling different text directions

  1. What property is used to switch the writing mode of the document?

    writing-mode

    1. What are the possible values for this property?

      horizontal-tb
      left-to-right inline direction and top-to-bottom block direction
      vertical-lr
      top-to-bottom inline direction and left-to-right block direction
      vertical-rl
      top-to-bottom inline direction and right-to-left block direction

  2. What is the logical, flow-relative equivalent to the width attribute?

    inline-size

  3. What is the logical, flow-relative equivalent to the height attribute?

    block-size

  4. What are the logical, flow-relative equivalents to the margin/padding/border -top, -right, -bottom, and -left properties?

    -block-start, -inline-end, -block-end and -inline-start

Overflowing content

  1. What is overflow?

    What happens when there is too much content to fit in a box

  2. What property allows you to control how an element overflows?

    overflow

    1. What is the default value for this property? Why?

      The default value is visible. This is to prevent data loss.

  3. How do you make overflow visible?

    overflow: visible

  4. How do you make overflow invisible?

    overflow: hidden

  5. How do you put scrollbars on overflowing content? What is the caveat to this?

    overflow: scroll

    The caveat is that the scrollbars will display even if the content isn't overflowing.

  6. How do you put scrollbars on overflowing content in only one direction?

    overflow-x: scroll for the horizontal direction and overflow-y: scroll for the vertical direction

  7. How do you set two different values for overflow, one for each direction?

    overflow is shorthand for overflow-x then overflow-y

  8. How do you make it so scrollbars only appear when the content overflows?

    overflow: auto

  9. What is a Block Formatting Context? What values of overflow create one?

    A BFC is when the content of a box becomes a layout of its own, and content cannot poke into or out of the container. This is created when you set overflow to any value other than visible.

CSS values and units

  1. What is another name for CSS values?

    CSS datatypes

  2. What is a <number>?

    Decimal numbers (negative, zero, or positive)

  3. What is a <dimension>? What values does this include?

    A <number> with a unit attached to it. Includes <length>, <angle>, <time>, and <resolution>.

  4. What values does the opacity property take?

    A <number> from 0 (transparent) to 1 (opaque).

  5. What is the difference between RGB and RGBA?

    RGBA takes opacity as a fourth parameter (alpha).

  6. What is HSL?

    Hue
    the base shade
    Saturation
    how much gray is in it from completely gray to no gray
    Light
    how much black is in it from completely black to completely white

Sizing items in CSS

  1. What is an element's intrinsic size?

    The natural size of the element before it is affected by CSS

  2. What is an <img>'s intrinsic size?

    An <img>'s intrinsic size is defined in the image file.

  3. What is a <div></div>'s intrinsic size?

    A <div></div> is only as tall as its content.

  4. What is an element's extrinsic size?

    The size given to it

  5. In what dimension does setting the size of an element risk overflow?

    The block dimension

  6. From what is the value of margins and padding when set in percentages calculated?

    The inline size

  7. What can you do when you know a box will contain a variable amount of content and you want it to always be at least a certain height?

    You can set a min-height.

  8. What can you do when you want an image to scale down when there's not enough space to display them at their intrinsic width, while making sure they don't become larger than that width?

    max-width: 100%

  9. What is 1vw equal to?

    1% of the viewport width

  10. What is 1vh equal to?

    1% of the viewport height

Images, media, and form elements

  1. What are replaced elements?

    Elements whose internal layouts are inaccessible to CSS

  2. How would you make it so a replaced element completely covers the box area while maintaining its aspect ratio?

    object-fit: cover

  3. How would you make it so a replaced element fits inside the box while maintaining its aspect ratio?

    object-fit: contain

  4. Do replaced elements stretch when put into flex or grid layouts?

    No.

  5. Write a basic form reset with comments explaining the purpose of each part.

    button,
input,
select,
textarea,
… {
  font-family: inherit;     /* because form elements don't inherit font styling by default */
  font-size: 100%;          /* same as above */
  box-sizing: border-box;   /* because box sizing is inconsistent across browsers */
  padding: 0; margin: 0;    /* good for consistency so you can add them back to your liking later */
}

  overflow: auto;           /* because IE puts scrollbars on <textarea></textarea>s by default */
}

Styling tables

  1. What is something that you should set on tables so you can size columns manually? Why is this necessary?

    table-layout: fixed

    This is necessary because by default, columns are sized according to their content which can lead to unpredictable results.

  2. What should you set on tables so table cells share borders?

    border-collapse: collapse

  3. How do you make it so letters are spaced apart?

    Use the letter-spacing property.

  4. What are zebra stripes? How do you set them?

    This is when the rows of a table alternate in color, making the table easier to read. You can do this using the :nth-child() pseudo-class with the even and odd keywords.

  5. How do you make it so text is centered in a table cell?

    text-align: center

Styling text

Fundamental text and font styling

  1. What property is used to set the font?

    font-family

  2. What are the web safe fonts and their respective caveats/things to remember about them?

    Arial
    It's good practice to use Helvetica as a preferred alternative
    Courier New
    Some OSs use Courier, so it's good practice to use both, with Courier New as the preferred alternative.
    Georgia
    No caveats
    Times New Roman
    Some OSs use Times, so it's good practice to use both, with Times New Roman as the preferred alternative.
    Trebuchet MS
    Not widely available on mobile OSs
    Verdana
    No caveats

  3. What is the general format of a font stack?

    1. Preferred font
    2. Alternative fonts (separated by commas)
    3. Web safe font
    4. Generic name

    e.g. "Trebuchet MS", Verdana, sans-serif

  4. What property is used to set the font size?

    font-size

  5. Is font size inherited?

    Yes.

  6. What is the default font size of <html></html>?

    16px

  7. How do you set how bold the text is? What two values for this are you most likely to use?

    Use the font-weight property. The two values you will probably use are normal and bold.

  8. What property can be used to make text all capitals? What values does this take?

    Use the text-transform property. Its values are as follows:

    • none
    • uppercase
    • lowercase
    • capitalize (first letter of every word is capitalized)
    • full-width (transforms all glyphs to be written in a fixed-width square, like a monospace font)

  9. What property can be used to put an underline on text? What values does this take?

    Use the text-decoration property.

    This is shorthand for:

    • text-decoration-line which takes:
      • none
      • underline
      • overline
      • line-through
    • text-decoration-style which takes:
      • solid
      • double
      • dotted
      • dashed
      • wavy
    • text-decoration-color

  10. How do you give text a drop shadow?

    Use the text-shadow property. It is formatted as follows:

    1. Horizontal offset (required)
    2. Vertical offset (required)
    3. Blur radius (defaults to zero if omitted)
    4. Color (defaults to black if omitted)

  11. How do you align text?

    Use the text-align property. Its values are as follows:

    • left
    • right
    • center
    • justify

  12. How do you set line height?

    Use the line-height property. Takes most length and size units, or a unitless value which acts as a multiplier (best option).

  13. How do you make it so words are spaced apart?

    Use the word-spacing property.

Styling lists

  1. How do you set the type of bullets to use for a list?

    Use the list-style-type property on the <ul></ul> or <ol></ol>.

  2. How do you set whether the bullets appear inside the list items or outside them before the start of each item?

    Use the list-style-position property on the <ul></ul> or <ol></ol> with a value of inside or outside.

  3. How do you set a custom image for the bullets?

    Use the list-style-image property with the url() function. Although, since this is limited in terms of controlling the position, size, etc. of the bullets, it's better to use the background properties.

  4. What is the shorthand for these properties?

    list-style with the values in any order

  5. How do you make an ordered list start on a number other than one?

    Use the start attribute on the <ol></ol> e.g. start="4" makes it count 4. 5. 6. etc.

  6. How do you make an ordered list count backwards?

    Use the boolean reversed attribute on the <ol></ol>. This continues to zero and into the negatives.

  7. How do you give specific numerical values to ordered list items?

    Use the value attribute on the <li></li>s.

Styling links

  1. How do you target an unvisited link?

    Use the :link pseudo-class.

  2. How do you target a visited link?

    Use the :visited pseudo-class.

  3. How do you target a link being hovered over?

    Use the :hover pseudo-class.

  4. How do you target a link being focused with keyboard controls or something else?

    Use the :focus pseudo-class.

  5. How do you target a link being clicked on?

    Use the :active pseudo-class.

    a:link {…}

a:focus {…}
a:hover {…}
a:active {…}

    To remember this, you can use the mnemonic LoVe Fears HAte.

CSS layout

Flexbox

  1. What is flexbox?

    It is a one-dimensional layout method for laying out items in rows and columns.

  2. How do you select which items are to be laid out as flexible boxes?

    Set display: flex on the flex container.

    1. What if you wanted to lay out inline items as flexible boxes?

      Set display: inline-flex on the flex container.

  3. What is the difference between the main axis and the cross axis?

    The main axis is the axis running in the direction the flex items are being laid out. The cross axis is the axis perpendicular to the main axis.

  4. How do you set the direction in which flex items are laid out?

    Use the flex-direction property. Its values are as follows:

    • column
    • row
    • column-reverse
    • row-reverse

  5. How do you make it so the flex items wrap when reaching the end of the main axis?

    Set flex-wrap: wrap on the container.

  6. How do you make it so the flex items are each 200px wide?

    Set flex: 200px on the container.

    1. How can you use this same property to instead set unitless proportion values on individual flex items to determine the space they take up?

      As an example, if one item has flex: 2 set and another has flex: 1 set, that first item will fill twice the amount of space as the second.

    2. How can you set a minimum width to be applied on a flex item before the unitless proportion value is applied?

      As an example, flex: 2 200px means that the flex item will be given a width of 200px and then the rest of the available space will be shared out according to the proportion values.

  7. What is the shorthand for the two properties from Questions 4 and 6?

    flex-flow

  8. How do you control where the flex items sit on the cross axis?

    Use the align-items property:

    align-items: stretch
    The default; items stretch to fill the parent in the cross direction
    align-items: center
    Items are centered along the cross axis
    align-items: flex-start
    Items are aligned at the start of the cross axis
    align-items: flex-end
    Items are aligned at the end of the cross axis

  9. How do you control where the flex items sit on the main axis?

    Use the justify-content property:

    justify-content: flex-start
    Items sit at the start of the main axis
    justify-content: flex-end
    Items sit at the end of the main axis
    justify-content: center
    Items sit at the center of the main axis
    justify-content: space-around
    Distributes items evenly along the main axis with some space at either end
    justify-content: space-between
    Distributes items evenly along the main axis with no space at either end

  10. How do you change the layout order of individual flex items?

    Use the order property. Flex items with greater order values appear later in the display order than items with lesser order values. You can use negative values to make items appear earlier than items with an order value of 0.

  11. Can you nest flex items (make a flex item a flex container)?

    Yes.

Grids

  1. What is CSS Grid Layout?

    It is a two-dimensional layout method for laying out items in rows and columns.

  2. What do you call the gaps between rows and columns?

    Gutters

  3. How do you select which items are to be laid out in a grid? What is the immediate effect of doing this?

    Set display: grid on the container. This has no effect until you add columns or rows.

  4. How would you arrange a grid into three 200px columns?

    Set grid-template-columns: 200px 200px 200px or grid-template-columns: repeat(3, 200px) on the container.

  5. How would you arrange a grid into three columns, one of which takes up twice as much of the available space as the other two?

    Set grid-template-columns: 2fr 1fr 1fr on the container.

  6. How would you make the gutters of a grid 20px wide?

    Set grid-gap: 20px and gap: 20px on the container.

    gap is the newer version, made with the intent of being able to be used for multiple layout methods. grid-gap will still work for some time, but you should put both just in case.

  7. How would you make the gutters between columns 20px wide and the gutters between rows 15px wide?

    Set grid-gap: 15px 20px and gap: 15px 20px on the container.

    grid-gap is shorthand for grid-row-gap then grid-column-gap (gap is shorthand for row-gap then column-gap).

  8. Using a function, how would you arrange a grid into three columns, each taking up the same amount of available space?

    Set grid-template-columns: repeat(3, 1fr) on the container.

  9. What is the difference between the explicit grid and the implicit grid? What are they analogous to in flexbox?

    The explicit grid consists of the rows and columns you set manually with grid-template-columns and grid-template-rows. The implicit grid is created when content is placed outside of that grid.

    This is analogous to the main and cross flexbox axes.

  10. How are implicit row and column tracks sized by default? How can you size them?

    By default, they are sized with auto, which means they are only large enough to fit their content. You can size them by setting the grid-auto-rows and grid-auto-columns properties on the container.

  11. How can you prevent overflow with implicit grid tracks that you sized?

    You can set grid-auto-rows: minmax(your min size, auto) on the container.

  12. How can you make a grid create as many columns of some minimum size as will fit in the container?

    You can set grid-template-columns: repeat(auto-fill, minmax(your min size, 1fr)) on the container.

  13. Write the CSS that places items in a grid so that:

    1. The <header></header> sits on the first row and spans columns 1 through 3
    2. The <article></article> sits on the second row and the second column
    3. The <aside></aside> sits on the second row and the first column
    4. The <footer></footer> sits on the third row and spans columns 1 through 3
    header {
  grid-column: 1 / 3;
  grid-row: 1;
}
article {
  grid-column: 2;
  grid-row: 2;
}
aside {
  grid-column: 1;
  grid-row: 2;
}
footer {
  grid-column: 1 / 3;
  grid-row: 3;
}

    1. What about for area-based placement?

      .container {
  grid-template-areas:
        "header header"
        "sidebar content"
        "footer footer";
}
header {grid-area: header;}
article {grid-area: content;}
aside {grid-area: sidebar;}
footer {grid-area: footer;}

    2. How do you specify an empty cell using area-based placement?

      Use a .

    3. How would you make the <article></article> three times as wide as the <aside></aside> using a line-based 12-column grid framework?

      header {

Floats

  1. If you have a box followed by text in the source code, How would you make it so the box floats to the left while the text wraps around it, like so:

    Set float: left on that box.

    1. What other values does this property take?

      • right
      • none
      • inline-start
      • inline-end

  2. How would you make it so some of the text moves down away from the box, like so:

    Set clear: left on that text.

    1. What other values does this property take?

      • both
      • right
      • none
      • inline-start
      • inline-end

  3. How would you make it so that, if you had a wrapper around the floated box and that first paragraph (first five lines), the rest of the text cleared the box so that the bottom of the wrapper box wraps the floating item and the wrapping content even if the content is shorter? For a visual aid:

    You want this:

    Not this:

    Describe three ways to accomplish this, and say which one is the preferred method.

    Clearfix:

    .wrapper::after {
  content: "";
  clear: both;
  display: block;                
}

    Using overflow (creates a BFC, but might create unwanted scrollbars or clip shadows):

Positioning

  1. What is static positioning? How do you set it?

    The default position of every element (positioned according to source order). You can set it with position: static.

  2. What is relative positioning? How do you set it?

    Positioning relative to the element's position in normal flow. You can set it with position: relative.

    1. How would you position an element to be 30px down and 20px to the right of its default, static position?

      Set top: 30px and left: 20px on the element.

  3. What is absolute positioning? How do you set it?

    Positioning relative to the element's containing block (nearest positioned ancestor). You can set it with position: absolute.

    1. How would you position an element to be 30px down and 20px to the right of the top-left corner of its containing block?

      Set top: 30px and left: 20px on the element.

    2. How do you identify the containing element of an absolutely positioned element?

      If all of the element's ancestor elements have a static position, then the containing block is the initial containing block, which contains the root element and has the dimensions of the viewport.

    3. What is the positioning context?

      The element that the absolutely positioned element is positioned relative to

    4. If you absolutely position an element, is the space that the element filled when it was in normal flow preserved?

      No, that gap is closed because the element was taken out of normal flow to sit on a separate layer.

    5. Does margin collapsing affect absolutely positioned elements?

      No, because it creates a BFC.

  4. In the case of overlapping, in what order are elements stacked by default?

    Elements appearing later in the source order overlap earlier elements.

  5. How do you change the order that overlapping elements are stacked?

    Use the z-index property, which works like the order property from flexbox.

  6. What is fixed positioning? How do you set it?

    It's like absolute positioning, except it's positioned relative to the browser viewport itself. You can set it with position: fixed.

  7. How do you make it so an element acts like it is relatively positioned until it is 10px from the top of the viewport, at which point it remains fixed at that position while scrolling?

    Set position: sticky and top: 10px.

Multiple-column layout

  1. How do you switch on multicol?

    Set either column-count (creates the specified number of columns) or column-width (creates as many columns of the specified width as can fit) on the container.

  2. How do you set the size of the gaps between columns?

    Set the column-gap property on the container.

  3. How do you put a rule in between columns?

    Set the column-rule property (takes the same values as border) on the container.

  4. How do you increase the space on either side of a rule?

    Increase the column-gap. The column-rule doesn't take up any space; it just lies across the gap.

  5. How do you make an element span across all of the columns? Some of the columns?

    Set column-span: all on the container. You can't make an element span only some columns; it's either none or all.

  6. How do you make it so an element doesn't fragment across columns?

    Set break-inside: avoid on that element (and the older property page-break-inside: avoid for best browser support).

Beginner's guide to media queries

  1. What is the basic media query syntax?

    @media media-type and (media-feature-rule) {rules}

  2. What are the possible media types?

    all, print, screen, and speech

  3. Write the media query that makes the body text blue if the screen's width is less than or equal to 600px.

    @media screen and (max-width: 600px) {
  body {
    color: blue;
  }
}

  4. Write the media query that makes the body text red if the screen's width is at least 600px.

    @media screen and (min-width: 600px) {
  body {
    color: red;
  }                
}

  5. Write the media query that makes the body text rebeccapurple if the screen is in landscape mode. What other values does this media feature take?

    @media screen and (orientation: landscape) {
  body {
    color: rebeccapurple;
  }                
}

    orientation also takes portrait as a value.

  6. Write the media query that makes the body text rebeccapurple if the user has the ability to hover over elements.

    @media (hover: hover) {
  body {
    color: rebeccapurple;
  }                
}

  7. Write the media query which tests that a user is using a mouse or a trackpad. What other values does this media feature take?

    @media (pointer: fine) {…}

    pointer also takes coarse (finger on a touch screen) and none (voice or keyboard controls).

  8. Write the media query which tests that the screen is both at least 600px wide and in landscape mode.

    @media screen and (min-width: 600px) and (orientation: landscape) {…}

  9. Write the media query which tests that the screen is at least 600px wide or in landscape mode.

    @media screen and (min-width: 600px), screen and (orientation: landscape) {…}

  10. Write the media query which tests that the screen is not in landscape mode.

    @media not all and (orientation: landscape) {…}

    The not operator negates the entire rest of the media query.

  11. What do you call the point at which a media query is introduced?

    A breakpoint
